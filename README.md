# 1_quaternions
Кватернионы в Python

## Installation

Создать виртуальное окружение

```bash
virtualenv venv
```

И активировать его
```powershell
# Для Windows
venv\Scripts\activate
```

```bash
# Для Linux
source venv/bin/activate
```

В корневой директории проекта выполнить

```bash
pip install .
```
